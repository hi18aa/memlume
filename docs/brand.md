# Memlume 品牌識別

## 名稱與概念

**Memlume** 由 **Memory**（記憶）與 **Lumen**（光）組成。圖標以深海軍藍圓環包圍五個記憶節點，節點相連形成可追溯的記憶圖譜；中央青藍光點與右下暖金弧線代表從可靠脈絡中取得清晰答案。

## 色票

| 名稱 | 色碼 | 用途 |
| --- | --- | --- |
| 深海軍藍 | `#0B1F33` | 外環、字標、節點輪廓 |
| 海灣藍 | `#103B5C` | 輔助深色區域 |
| 青藍 | `#27C2D8` | 節點連線、主要光感 |
| 淺青 | `#9AF4F4` | 高光與輔助節點 |
| 暖金 | `#F4B75B` | 少量強調與光弧 |
| 霧白 | `#DFFCFF` | 中央光芒 |

## 檔案與使用原則

- `assets/logo/memlume-logo.svg` 是權威主檔：透明背景、可編輯、包含圖標與橫式字標。
- `assets/logo/memlume-logo.png` 與 `assets/logo/favicon.png` 都是由 SVG 圖標渲染的產物；**不得手動修改**，需重新由 SVG 產生。
- 請保留圖標、字標及其比例；不得拉伸、旋轉、重繪節點或改變既有色票。

## 最小尺寸與底色

| 用法 | 最小尺寸 | 建議使用方式 |
| --- | --- | --- |
| 橫式 Logo | 寬 180 px | 透明背景；淺底直接使用主檔。 |
| 僅圖標 | 64 × 64 px | favicon 與小型介面使用 `favicon.png`。 |
| 印刷 | 橫式寬 32 mm | 優先使用 SVG 或其他向量輸出。 |

- **淺色底**：直接使用主檔；深海軍藍字標與外環提供足夠對比。
- **深色底**：使用 SVG 時，請將字標與外環改為霧白 `#DFFCFF` 的反白版本；青藍與暖金節點維持原色。至少預留圖標外環寬度一半的淨空，避免被相近深藍背景吞沒。
