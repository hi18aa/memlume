# Memlume 品牌識別

## 名稱與概念

**Memlume** 由 **Memory**（記憶）與 **Lumen**（光）組成。圖標以深海軍藍圓環包圍五個記憶節點，節點相連形成可追溯的記憶圖譜；中央青藍光點與右下暖金弧線代表從可靠脈絡中取得清晰答案。

## 色票

| 名稱 | 色碼 | 用途 |
| --- | --- | --- |
| 深海軍藍 | `#0B1F33` | 外環、字標、節點輪廓 |
| 海灣藍 | `#103B5C` | 輔助深色區域 |
| 青藍 | `#27C2D8` | 節點連線、主要光感 |
| 淺青 | `#9AF4F4` | 高光與輔助節點 |
| 暖金 | `#F4B75B` | 少量強調與光弧 |
| 霧白 | `#DFFCFF` | 中央光芒 |

## 檔案與使用原則

- `assets/logo/memlume-logo.svg` 是權威主檔：透明背景、可編輯、包含圖標與橫式字標；字標是原創向量 path，不依賴系統或外部字型。
- SVG 的完整 lockup 使用 `viewBox="0 0 760 192"`；方形 mark 使用同一份幾何的圖標裁切 `viewBox="18 18 156 156"`，字標自然落在裁切範圍外。
- `assets/logo/memlume-logo.png`（512 × 512）與 `assets/logo/favicon.png`（64 × 64）皆由該方形裁切以透明背景渲染；**不得手動修改**，需重新由 SVG 產生。
- 可重現輸出：保留 SVG 原始節點幾何，以瀏覽器或 SVG renderer 指定方形裁切與目標尺寸渲染，再檢查 alpha 四角為透明；不得以截圖、補繪或點陣編修取代。
- 請保留圖標、字標及其比例；不得拉伸、旋轉、重繪節點或改變既有色票。

## 最小尺寸與底色

| 用法 | 最小尺寸 | 建議使用方式 |
| --- | --- | --- |
| 橫式 Logo | 寬 180 px | 透明背景；使用同一 SVG 主檔。 |
| 僅圖標 | 64 × 64 px | 淺底小型介面使用 `favicon.png`。 |
| 印刷 | 橫式寬 32 mm | 優先使用 SVG 或其他向量輸出。 |

- **淺色底**：同一 SVG 主檔會使用深海軍藍字標與外環，適合白色與淺色表面。
- **深色底**：同一 SVG 主檔透過內嵌 `prefers-color-scheme: dark` 自動將字標、外環與節點輪廓切為霧白 `#DFFCFF`；不需手動修改來源。青藍與暖金節點維持原色。至少預留圖標外環寬度一半的淨空。
- **PNG／favicon**：兩者是透明的淺色模式方形 mark，僅用於淺色介面；深色介面應使用可回應色彩模式的 SVG，而非手動改 PNG。
