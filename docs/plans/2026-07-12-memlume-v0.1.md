# Memlume v0.1 實作計畫

> **Required sub-skill:** Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 發布可在本機安裝、以 CLI 與 MCP 共用結構化記憶的 Memlume v0.1.0。

**Architecture:** 以 TypeScript pnpm workspace 建立 contracts、SQLite database 與 core packages；daemon 是唯一寫入者，CLI 與 MCP 皆透過 localhost API 使用它。第一版以 SQLite FTS5 檢索，不加入向量資料庫、UI 或雲端服務。

**Tech Stack:** Node.js LTS、TypeScript、pnpm、Zod、better-sqlite3、Express、MCP TypeScript SDK、Commander、Vitest、SQLite FTS5。

## 建議子代理分工

| 時機 | 子代理任務 | 邊界 |
| --- | --- | --- |
| Phase 0 | 建立三個 Logo 候選與資產規格 | 僅修改 `assets/logo/`、`docs/brand.md` |
| Phase 1 | 審查 contracts 與 SQLite migration | 僅修改 `packages/contracts/`、`packages/database/`、相關測試 |
| Phase 2–3 | 實作與測試 CLI/MCP 封裝 | 不修改 schema、migration 或 daemon API |
| Phase 4 | 撰寫英文 README、翻譯與三語一致性檢查 | 僅修改 `README.md`、`docs/README.*` |

每一項子代理工作完成後，由主代理執行該 package 的測試與完整驗證；不要平行修改相同檔案或同一 API 契約。
